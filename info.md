# Инструкция для работы с Git.

* git init - инициализация локального репозитория
* git status - получить информацию от git о его текущем состоянии
* git add - добавить файл или файлы для отслеживания
* git commit -m "сообщение" - создание коммита
* git log - отобразить истолию логов (коммитов)
* git checkout (4-6 символа) - переход к конкретному коммиту
* git checkout master - вернуться к актуальному состоянию
* git diff - увидеть разницу между текущим файлом и закоммиченым
